# MAKI – Aplicativo Web para Restaurante Japonés 

**MAKI** es una aplicación web diseñada para digitalizar la gestión integral de un restaurante japonés. Permite a los clientes realizar pedidos de forma intuitiva y al personal gestionar la operación de manera eficiente y escalable.

Este proyecto fue desarrollado como parte de la asignatura **Desarrollo Web** en la **Pontificia Universidad Javeriana – Sede Bogotá**.

---

## Descripción del Proyecto
El objetivo principal de MAKI es modernizar la experiencia del usuario y la logística interna, reemplazando los sistemas manuales (como llamadas telefónicas) por una solución digital centralizada.
